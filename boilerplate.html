Boilerplate code

To review the HTML boilerplate code. Two types of tags must be known, those with open and close tags, which are the standard and those without closing tags known as void elements

Doctype

The document type declaration tells the browser what version of HTML the page is using. In modern HTML, the declaration is written as <!DOCTYPE html>.

Older versions of HTML used longer declarations. For example, HTML 4 could use: <!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Transitional//EN" "http://www.w3.org/TR/html4/loose.dtd">.

XHTML also used a longer declaration, such as: <!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.1//EN" "http://www.w3.org/TR/xhtml11/DTD/xhtml11.dtd">.

Html

The <html></html> element is the root element of an HTML document. It is a standard element which contains the main HTML structure of the page, except for the <!DOCTYPE html> declaration.

The <html> element can have a lang attribute. For example, lang="en" tells the browser that the page language is English.

Head

The <head></head> element contains metadata about the current webpage. Metadata is information about the page that is not usually shown directly on the page.

Elements commonly found inside the <head> include <title>, <meta>, and <link>.

  1. The <title></title> element sets the title of the webpage. This title usually appears in the browser tab.

  2. The <meta> element can give the browser extra information about the page, such as the character set, viewport settings, and page description.

  3. The <link> element is often used to connect an HTML file to another file, such as a CSS stylesheet.

Body

The <body></body> element contains the visible content of the webpage, such as headings, paragraphs, images, and links.
